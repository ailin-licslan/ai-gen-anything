# AI 双语视频自动生成系统

全自动语音/文本 → 双语字幕视频生成工具
支持：音频识别、自动改写、TTS 合成、中英双语字幕、背景音乐、视频合成

## ✨ 功能
- 上传音频/文本/文档/表格 → 自动生成对话视频
- 录音 → 自动生成视频
- 默认 demo 一键生成
- LLM 文本改写（英文 → 英文，只改写不翻译）
- 中英双语字幕自动生成
- 智能配音（多角色）
- 背景音乐 + 动态画面效果

## 🧱 技术栈
- FastAPI
- Edge-TTS
- FFmpeg
- Google 翻译
- OpenRouter LLM
- 阿里 ASR
